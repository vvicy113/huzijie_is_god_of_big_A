# 选股器体系

## 接口

```java
interface StockSelector {
    int getCode();    // 唯一标识码 → SelectorConstants 中定义
    List<String> select(List<String> candidates, LocalDate date);
}

interface StockSelectorRegistry {
    void register(int code);              // 按常量码注册
    void register(StockSelector selector); // 直接注册实例
    StockSelector build();                // 构建责任链
}

class StockSelectorChain implements StockSelector {
    // 按顺序串联多个 Selector，依次过滤
    // 自身也是 Selector，可嵌套
}
```

## 使用方式

```java
// 场景1：短线动量选股
StockSelectorRegistry reg = new DefaultSelectorRegistry("动量短线");
reg.register(ALL_MAIN_BOARD);          // 全市场
reg.register(VOL_20_1P5);              // 放量
reg.register(MOMENTUM_10_5_30);        // 10日动量
StockSelector chain = reg.build();

// 场景2：底部反转选股
StockSelectorRegistry reg2 = new DefaultSelectorRegistry("底部反转");
reg2.register(ALL_MAIN_BOARD);
reg2.register(POS_RANK_LOW_20);        // 20日低位
reg2.register(BOLL_LOWER_20_2);        // 布林下轨
reg2.register(LOWER_SHADOW_2);         // 长下影线

// 执行
List<String> stocks = chain.select(List.of(), LocalDate.parse("2024-03-15"));
```

## 工厂扩展

```java
class MyRegistry extends DefaultSelectorRegistry {
    @Override
    protected StockSelector createFromCode(int code) {
        return switch (code) {
            case 9999 -> new MyCustomSelector();
            default -> super.createFromCode(code);
        };
    }
}
```
