# 策略与引擎设计

## Strategy 接口

```java
interface Strategy {
    String getName();                          // 策略名称
    StockSelector getStockSelector();           // 选股器
    void evaluate(StrategyContext ctx);         // 评估 → SignalBoard
    default void onReset() {}                   // 回测前重置
}
```

## StrategyContext

```java
class StrategyContext {
    List<String> candidates();                 // 候选股票列表
    Map<String, List<KLine>> klineMap();       // stockCode → 历史K线
    Map<String, Position> positions();         // stockCode → 持仓
    double availableCash();                    // 可用资金
    LocalDate date();                          // 当前交易日
    SignalBoard signalBoard();                 // 信号板

    // 快捷方法
    KLine todayKLine(String code);             // 今日K线
    List<KLine> history(String code);          // 历史K线
    Position position(String code);            // 持仓状态
    boolean isHolding(String code);            // 是否持仓
    void signal(String code, TradeAction action, double score); // 添加信号
}
```

## TradeSignal / SignalBoard

```java
record TradeSignal(String code, TradeAction action, double score)
    implements Comparable  // 默认按 score 降序

class SignalBoard {
    void add(TradeSignal);               // 添加/更新信号
    void remove(String stockCode);       // 按代码移除
    TradeSignal first();                 // 最高分
    List<TradeSignal> topN(int n);       // 前 N 名
    List<TradeSignal> all();             // 全部
    SignalBoard filterBuy(double min);   // 只保留 BUY 且 score>min
    void clear();                        // 清空
}
```

内部：TreeSet 按 score 降序 + HashMap 按 code 快速查找，双索引。

## 回测引擎（设计态）

```
BacktestEngine.run(strategy, from, to, initialCapital)

每天循环:
  1. StockSelector.select(pool, date)              → 候选池
  2. 加载每只候选的历史 K 线                         → klineMap
  3. 构建持仓 Map                                   → positions
  4. 创建 StrategyContext                           → ctx
  5. strategy.evaluate(ctx)                         → 填充 SignalBoard
  6. 遍历 SignalBoard (按 score 降序):
     - BUY:  资金够 → 买入（全仓？第N只？等分？）
     - SELL: 清仓该股票
  7. 风控检查: 止损止盈（待实现）
  8. 记录每日权益
```

## 策略实现示例（待实现）

```java
class MomentumBreakoutStrategy implements Strategy {
    private final StockSelector selector;
    private final StockComparator comparator;

    MomentumBreakoutStrategy() {
        DefaultSelectorRegistry reg = new DefaultSelectorRegistry("动量突破");
        reg.register(ALL_MAIN_BOARD);        // 全市场
        reg.register(VOL_20_1P5);            // 放量
        reg.register(MOMENTUM_10_5_30);      // 动量
        this.selector = reg.build();
    }

    public StockSelector getStockSelector() { return selector; }

    public void evaluate(StrategyContext ctx) {
        for (String code : ctx.candidates()) {
            if (ctx.isHolding(code)) {
                // 仍符合条件，继续持有
                ctx.signal(code, HOLD, 50);
                continue;
            }
            KLine today = ctx.todayKLine(code);
            double score = computeScore(code, today);
            if (score > 70) ctx.signal(code, BUY, score);
        }
    }
}
```
