# 系统架构

## 分层视图

```
┌──────────────────────────────────────────────────────────┐
│  CLI 层 (cli/)                                           │
│  ConsoleUI → BacktestMenu / AnalysisMenu / DataMenu      │
├──────────────────────────────────────────────────────────┤
│  业务层                                                   │
│  ┌─────────────┐  ┌──────────────┐  ┌──────────────────┐│
│  │ backtest/    │  │ analysis/    │  │ selector/        ││
│  │ engine/      │  │ fetcher/     │  │ 23个选股器        ││
│  │ strategy/    │  │ analyzer/    │  │ Chain + Registry ││
│  │ metrics/     │  │ llm/         │  └──────────────────┘│
│  │ report/      │  └──────────────┘                      │
│  └─────────────┘                                         │
├──────────────────────────────────────────────────────────┤
│  数据层                                                   │
│  ┌──────────────────────────────────────────────────────┐│
│  │ db/                                                  ││
│  │ DatabaseManager → KLineRepository → CsvDataLoader    ││
│  │ CsvImporter → ProjectPaths                           ││
│  └──────────────────────────────────────────────────────┘│
├──────────────────────────────────────────────────────────┤
│  模型层 (model/)                                          │
│  KLine / TradeSignal / Position / BacktestResult / ...   │
├──────────────────────────────────────────────────────────┤
│  常量层 (constants/)                                      │
│  SelectorConstants                                       │
├──────────────────────────────────────────────────────────┤
│  注册表层 (registry/)                                     │
│  StockSelectorRegistry → DefaultSelectorRegistry         │
└──────────────────────────────────────────────────────────┘
```

## 包职责

| 包 | 职责 | 状态 |
|------|------|------|
| `cli/` | 控制台交互菜单 (ConsoleUI / BacktestMenu / AnalysisMenu / DataMenu) | 部分完成 |
| `model/` | 纯数据对象，Lombok @Data/@Builder (KLine / Position / TradeSignal / ...) | 完成 |
| `db/` | SQLite 连接管理、K线数据读写、CSV导入 | 完成 |
| `backtest/loader/` | 从 SQLite 加载日K线数据 | 完成 |
| `backtest/strategy/` | 策略接口 + SignalBoard + StrategyContext + StrategyRegistry | 接口完成，无实现 |
| `backtest/engine/` | 回测引擎，逐日执行买卖 | 未实现 |
| `backtest/metrics/` | 绩效指标计算 (收益率/回撤/夏普) | 完成 |
| `backtest/report/` | 回测报告格式化输出 | 完成 |
| `selector/` | 23 个选股器 + StockSelector 接口 + StockSelectorChain 责任链 | 完成 |
| `registry/` | StockSelectorRegistry 接口 + DefaultSelectorRegistry 实现 | 完成 |
| `constants/` | SelectorConstants 常量码定义 + code→desc 映射 | 完成 |
| `analysis/fetcher/` | 东方财富 API 联网抓取 (StockInfo/行业/概念/分时) | 完成 |
| `analysis/analyzer/` | Pearson 相关性 + 领涨跟风检测 + 综合分析 | 完成 |
| `analysis/llm/` | Claude API 调用，生成自然语言分析报告 | 完成 |
