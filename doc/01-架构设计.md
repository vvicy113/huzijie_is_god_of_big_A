# 系统架构

## 分层视图

```
run.properties  ← 运行配置（数字枚举）
      │
      ▼
App.main() → ConsoleUI → 读取配置 → 自动执行

───────────────────────────────────────────────
│ CLI 层 (cli/)                                     │
│ ConsoleUI → BacktestMenu / AnalysisMenu / DataMenu │
├───────────────────────────────────────────────────┤
│ 业务层                                             │
│ ┌──────────────┐ ┌──────────────┐ ┌─────────────┐│
│ │ backtest/    │ │ analysis/    │ │ selector/    ││
│ │ engine/      │ │ fetcher/     │ │ 23个选股器   ││
│ │ executor/    │ │ analyzer/    │ │ comparator/  ││
│ │ strategy/    │ │ llm/         │ │ 5个打分器    ││
│ │ metrics/     │ └──────────────┘ └─────────────┘│
│ │ report/      │ ┌──────────────┐ ┌─────────────┐│
│ └──────────────┘ │ config/      │ │ registry/   ││
│                  │ BacktestConfig│ │ SelectorReg ││
│                  │ RunConfig    │ └─────────────┘│
├───────────────────────────────────────────────────┤
│ 数据层                                             │
│ db/ → DatabaseManager / KLineRepository /          │
│       CsvImporter / ProjectPaths                   │
├───────────────────────────────────────────────────┤
│ 模型层 (model/)                                    │
│ KLine / TradeSignal / TradeRecord / Position / ...  │
├───────────────────────────────────────────────────┤
│ 常量层 (constants/)                                │
│ SelectorConstants / ComparatorConstants            │
└───────────────────────────────────────────────────┘
```

## 包职责

| 包 | 职责 | 状态 |
|------|------|------|
| `cli/` | 读取 run.properties 自动执行 | ✅ |
| `model/` | 纯数据对象 | ✅ |
| `db/` | SQLite + K线读写 + 元信息 | ✅ |
| `backtest/engine/` | 编排：选股→打分→风控→执行→报告 | ✅ |
| `backtest/executor/` | TradeExecutor 接口 + 等分资金实现 | ✅ |
| `backtest/strategy/` | Strategy + SignalBoard + Context + Registry | ✅ |
| `backtest/loader/` | 从SQLite加载K线 | ✅ |
| `backtest/metrics/` | 收益/回撤/夏普 | ✅ |
| `backtest/report/` | 报告格式化 | ✅ |
| `selector/` | 23个选股器 + 责任链 | ✅ |
| `comparator/` | 5个打分器 + 综合加权 | ✅ |
| `registry/` | 选股器注册表 | ✅ |
| `config/` | BacktestConfig + RunConfig | ✅ |
| `constants/` | 选择器/打分器常量码 | ✅ |
| `analysis/` | 关联分析 + LLM | ✅ |
