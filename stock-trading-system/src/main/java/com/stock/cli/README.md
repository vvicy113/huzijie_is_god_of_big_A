# cli — 命令行交互界面

基于 `Scanner` 的控制台菜单驱动 UI，不依赖任何框架。

## 文件

| 文件 | 功能 |
|------|------|
| `ConsoleUI.java` | 主菜单，提供回测 / 关联分析 / 数据管理三个入口 |
| `BacktestMenu.java` | 回测子菜单 — 输入股票代码 → 选策略 → 执行回测 → 输出报告 |
| `AnalysisMenu.java` | 关联分析子菜单 — 输入股票代码 → 联网抓取 → 量化分析 → LLM 综合解读 |
| `DataMenu.java` | 数据管理子菜单 — 导入日K线到数据库 / 导入分时数据到数据库 / 查看数据库统计 |
